Проекты

Проект по анализу пользовательских сессий Tele2

Стек: Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy), Exel - На основании разведочного анализа произвела выбор полезных данных из датасета, сформулировала гипотезу, механизм. - Подобрала статистический метод для сравнения конверсий между выборочной и генеральной группами (Z-score), а также реализовала проверку для всех возможных групп на языке Python. - По результатам проверки выявила статистически значимо отстающую по конверсиям группу клиентов.

Проект по аналитике компании велопроката Divvy (США, Чикаго)

Провела разведывательный aнализ (EDA) на большом массиве данных, анализ зависимости числа поездок от дня недели, по итогам анализа был сделан вывод о возможном наличии маятниковых станций (станций, где велосипеды используются людьми преимущественно для того чтобы добраться от дома до работы/учебы и обратно), а также занималась анализом сезонности числа поездок. Исследовала вопрос окупаемости одного велосипеда, оценивая стоимость велосипеда, комплектующих (а также километраж, после которого их необходимо менять), стоимость ТО, стоимость аренды велосипеда, подписки и. т.
